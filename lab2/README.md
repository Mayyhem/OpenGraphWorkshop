# Lab 2: GitHub Data Collection

Three progressive sub-labs that build a real-world graph by collecting data from the GitHub API.

| Sub-lab | Description |
|---------|-------------|
| [0_collecting-an-edge/](0_collecting-an-edge/) | Collect contributors from a single GitHub repository |
| [1_expanding-the-graph/](1_expanding-the-graph/) | Expand to multiple repositories within an organization |
| [2_hybrid-edges/](2_hybrid-edges/) | Link GitHub users to X (Twitter) profiles with cross-source edges |

Each sub-lab includes data collection scripts in Bash, Python, and PowerShell, along with pre-collected sample data so you can follow along without making API calls.
